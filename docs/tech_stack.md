# Tech Stack: Pelos Match

## 1. Tecnologías
**Ahora:**
* HTML5: Utilizado para la estructura semántica de la aplicación.
* Tailwind CSS: Framework principal para el diseño y estilizado mediante clases utilitarias, permitiendo una maquetación rápida de las tarjetas de "swipe".
* Google Fonts: Implementación de fuentes redondeadas para reforzar la identidad visual amigable y animalista.
* JavaScript: Se utilizará para gestionar la manipulación del DOM y los eventos de interacción.
* LocalStorage: Para el almacenamiento local de los "Matches" y preferencias del usuario sin necesidad de un backend inicial.

**Futuro:**
* React.js
* Firebase
* Base de Datos

## 2. Herramientas de Desarrollo y Control de Versiones
* Git: Para el control de versiones.
* GitHub: Repositorio remoto para la colaboración en equipo y despliegue del proyecto.
* IA: Utilizada como "Pair Programmer" para agilizar la escritura de código.
* Visual Studio Code: Como editor de texto.
