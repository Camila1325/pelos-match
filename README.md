# Pelos Match 🐾
**"Encuentra a tu compañero de vida, un pelito a la vez."**

Esta es una aplicación web/móvil diseñada para conectar animales, en hogares de paso o centros de refugios, con adoptantes responsables mediante una interfaz interactiva de emparejamiento.
